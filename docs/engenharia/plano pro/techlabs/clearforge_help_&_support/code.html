Como ativar minha chave PRO?

Vá em Configurações > Conta e insira o código de 25 dígitos recebido por e-mail. A ativação é instantânea e requer conexão com a internet.

A licença Padrão permite 1 dispositivo, enquanto a licença PRO permite até 3 ativações simultâneas.
